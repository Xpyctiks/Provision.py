Automatic site provisioning system:
Add new sites:
Step 1:

Upload a ZIP archive with at least following files inside:

public/: - (REQUIRED!) folder with web scripts.

site_name.crt - (REQUIRED!) certificate file. name should be full domain name + .crt (site.com.crt for example)

site_name.key - (REQUIRED!) certificate key file. name should be full domain name + .key (site.com.key for example)

htpasswd - (REQUIRED!) a file which contains password hash for future access to admin panel


You can upload multiple files at the same time.
All files and folders inside the archive will be unpacked to the server's web folder.

"rewrite" file will be deleted after unpacking.

Certificate files will be moved from the web folder to web server's folder.
