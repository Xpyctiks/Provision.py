крок 1:

Завантажте ZIP архів,в якому має бути мінімум наступне:

public/: - (ВАЖЛИВО!) головна папка зі скриптами.

site_name.crt - (ВАЖЛИВО!) файл сертифікату, ім'я має бути повним доменом сайту + .crt в кінці(site.com.crt наприклад)

site_name.key - (ВАЖЛИВО!) файл ключа, ім'я має бути повним доменом сайту + .key в кінці(site.com.key наприклад)

htpasswd - (ВАЖЛИВО!) файл з хешем паролю для доступа до адмінки


Можна завантажити одразу багато файлів.
Всі файли та папки будуть розпаковані на сервері в веб папку.

Файл "rewrite" буде видалено після розпаковки.

Файли сертифікатів будуть переміщені до папки веб серверу.
